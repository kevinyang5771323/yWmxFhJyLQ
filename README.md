# 二手交易管理系统 java415

## 项目概述

二手交易管理系统是一个基于Java和Vue的前后端分离的web网站，采用B/S架构和MVC设计模式，提供二手物品交易相关的各项功能。

## 技术栈

- **后端：**
  - Java
  - SpringBoot
  - MyBatis
  - Mysql
- **前端：**
  - Vue
- **开发工具：**
  - IDEA
  - JDK 1.8
  - Maven
  - Mysql 8.0

## 功能模块

### 前台功能

1. 首页：支持全部查询、分类查询、关键字搜索。
2. 发布物品：用户可以发布自己的二手物品。
3. 查看消息：查看与交易相关的通知信息。
4. 个人中心：
   - 编辑个人信息
   - 编辑收货地址
5. 状态查询：
   - 我发布的物品
   - 我下架的物品
   - 我收藏的物品
   - 我卖出的物品
   - 我买到的物品

### 后台功能

1. 交易物品管理：
   - 上线的二手物品（支持违规下架）
   - 下架的二手物品（支持永久删除）
2. 订单信息管理：管理用户的订单信息，支持删除操作。
3. 用户信息管理：
   - 正常用户（支持封号操作）
   - 违规用户（支持解封操作）
   - 管理员（支持添加管理员）

## 系统角色

系统包含以下角色：
- 普通用户：可以发布物品、查看消息、管理个人中心。
- 管理员：负责交易物品管理、订单信息管理、用户信息管理。
